Capítulo 2 – Ítem iii: Producto de dos polinomios

Solución

  1. \((x+y)(x^{2}+y^{2}) = x^{3}+x^{2}y+xy^{2}+y^{3}\)
  2. \((2a+b)(3a-2b) = 6a^{2}-ab-2b^{2}\)
  3. \((1-x)(1-y) = 1-x-y+xy\)
  4. \((2x-6y)(x^{2}-2xy) = 2x^{3}-10x^{2}y+12xy^{2}\)
  5. \((x^{2}+3x^{2}y)(-3xy^{2}+4xy^{3}) = -3x^{3}y^{2}-5x^{3}y^{3}+12x^{3}y^{4}\)
  6. \((4x+y)(-2x-5xy) = -8x^{2}-20x^{2}y-2xy-5xy^{2}\)
  7. \((6a-5b)(2b+7a) = 42a^{2}-23ab-10b^{2}\)
  8. \((a+b+1)(a-b) = a^{2}-b^{2}+a-b\)
  9. \((2a-3ab+b^{2})(b-b^{2}) = 2ab-5ab^{2}+3ab^{3}+b^{3}-b^{4}\)
  10. \((5x^{2}y+2xy^{2}-3xy)(x-y^{2}) = 5x^{3}y-3x^{2}y-5x^{2}y^{3}+2x^{2}y^{2}-2xy^{4}+3xy^{3}\)
  11. \((m^{2}+n^{2}-mn)(2m-3n) = 2m^{3}-5m^{2}n+5mn^{2}-3n^{3}\)
  12. \((-3xy-2xy^{2})(xy^{2}-5xy) = 15x^{2}y^{2}+7x^{2}y^{3}-2x^{2}y^{4}\)
  13. \((2p^{2}q+3pq^{11}-5pq^{4})(-3pq+2p) = -6p^{3}q^{2}+4p^{3}q-9p^{2}q^{12}+6p^{2}q^{11}+15p^{2}q^{5}-10p^{2}q^{4}\)
  14. \((x^{2}+1)(x^{2}-1) = x^{4}-1\)
  15. \((a+b)(a-b) = a^{2}-b^{2}\)
  16. \((x+4)(x-6) = x^{2}-2x-24\)
  17. \((a^{2}+5)(a^{2}+7) = a^{4}+12a^{2}+35\)
  18. \((2p-4)(2p+7) = 4p^{2}+6p-28\)
  19. \((2x-3y-4z)(x+y+z) = 2x^{2}-xy-2xz-3y^{2}-7yz-4z^{2}\)
  20. \((x^{2}+y^{2}-z^{2})(2x-3y-4z)=2x^{3}-3x^{2}y-4x^{2}z+2xy^{2}-2xz^{2}-3y^{3}-4y^{2}z+3yz^{2}+4z^{3}\)
  21. \((a+1)(a^{n}+a^{n+1}+a^{n+2})=a^{n}+2a^{n+1}+2a^{n+2}+a^{n+3}\)
  22. \((a-1)(a^{n-1}+a^{n}+a^{n+1}) = a^{\,n+2}-a^{\,n-1}\)
  23. \((u-v)(u^{2}-3uv+v^{2}) = u^{3}-4u^{2}v+4uv^{2}-v^{3}\)
  24. \((x+y)(x^{2}+2xy+y^{2}) = x^{3}+3x^{2}y+3xy^{2}+y^{3}\)
  25. \((-3x+y^{2})(x^{2}-xy-y) = -3x^{3}+3x^{2}y+x^{2}y^{2}-xy^{3}+3xy-y^{3}\)
  26. \((2y+3x)(x^{2}-xy+2y^{2}) = 3x^{3}-x^{2}y+4xy^{2}+4y^{3}\)
  27. \((-3x-2y+z)(x+y-3z) = -3x^{2}-5xy+10xz-2y^{2}+7yz-3z^{2}\)
  28. \((x-y)(x^{2}+xy+y^{2}) = x^{3}-y^{3}\)
  29. \((x+y)(x^{2}-xy+y^{2}) = x^{3}+y^{3}\)
  30. \((a+b)(a^{4}-a^{3}b+a^{2}b^{2}-ab^{3}+b^{4}) = a^{5}+b^{5}\)
  31. \((a-b)(a^{3}+a^{2}b+ab^{2}+b^{3}) = a^{4}-b^{4}\)
  32. \((x+y)(x^{n-1}+x^{n-2}+x^{n-3}) = x^{n}+(1+y)x^{n-1}+(1+y)x^{n-2}+yx^{n-3}\)
  33. \((p^{2}-q^{2})(p^{\,n}-p^{\,n}q^{\,n}-q^{\,n})=p^{\,n+2}-p^{\,n+2}q^{\,n}-p^{2}q^{\,n}-p^{\,n}q^{2}+p^{\,n}q^{\,n+2}+q^{\,n+2}\)
